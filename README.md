# Calculadora em Java simples

Este é meu primeiro projeto em Java, desenvolvido para praticar a logica de programação. 

#Funcionalidades
  -Soma
  -Subtração
  -Multiplicação
  -Divisão (com validade de divisao por zero) 

#Tecnologias utilizadas 
  -Java
  -Estruturas condigionais (if/else e swith case)

#Aprendizados
Neste projeto pratiquei:
- Entrada de dados com Scanner
- Uso de switch case
- Estruturas condicionais
- Logica de programação
